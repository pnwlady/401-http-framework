# waka-flocka-frame
​
<strong>Waka Flocka Frame http Framework</strong>
​
How to use the WWF framework
=====================
Installation
-------------
```npm install waka-flocka-frame```
​
Then require ('waka-flocka-frame') in your project.
​
```var wwf = require('waka-flocka-frame')```
